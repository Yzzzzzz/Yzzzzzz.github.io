hugo 自学教程

快速开始

mac系统可以使用 brew 进行快速安装 > brew install hugo

如果你可以的话，可以参考hugo的官方文档，我在这里只是把简单的安装操作流程梳理了一下，仅供大家参考

生成站点

执行一下命令，新生成的站点目录文件夹就会在命令行当前目录生成

hugo new site blog

创建新的博客文章

执行以下命令，会在blog/content/post/first.md 生成文件

hugo new post/first.md
